az cognitiveservices
Manage Azure Cognitive Services accounts.
This article lists the Azure CLI commands for Azure Cognitive Services account and subscription management only. Refer to the documentation at
https://learn.microsoft.com/azure/cognitive-services/
for individual services to learn how to use the APIs and supported SDKs.
