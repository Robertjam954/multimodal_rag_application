A multimodal RAG tutor that cites its sources.

What it does

Retrieval you can trust

Every capability is wired to real Azure services - this is a running system, not a mockup.

📄

PDF ingestion

Drop in a document - it's parsed, chunked, embedded with text-embedding-3-large, and indexed into Cosmos DB, ready to query in seconds.

🔎

Cosmos vector search

Integrated NoSQL vector search (diskANN, cosine, 3072-dim) retrieves the most relevant passages via VectorDistance().

🧭

Multi-agent QA

A Router -> Retriever -> Answerer -> Verifier loop. The Verifier gates claims against evidence so answers stay grounded.

📌

Page-level citations

Answers carry inline [doc:page] markers that deep-link back to the exact source passage.

🔐

Keyless / Entra ID

No secrets in the app. Every call - Foundry, Cosmos, Blob - authenticates through managed identity and Entra ID.

⚡

Streaming chat

Tokens, citations, and the verifier verdict stream over SSE for a responsive, transparent answer experience.

How it works

From upload to grounded answer

The full path a question travels, end to end.

01

Ingest

PDF parsed & chunked, each chunk embedded and upserted to Cosmos.

02

Retrieve

Query embedded, nearest passages pulled via Cosmos vector search.

03

Answer

gpt-4o-mini drafts an answer with inline citation markers.

04

Verify

Verifier checks each claim against the retrieved evidence.

05

Stream

Answer, citations & verdict stream back to the browser.

Under the hood

Architecture

A Quart backend (with the React frontend bundled in) on Azure Container Apps, backed entirely by managed Azure services.

Browser (React 19 + Vite)
  │ HTTPS / SSE
Quart backend · Azure Container Apps (keyless, Entra ID)
  ├─ Azure AI Foundry — gpt-4o-mini (chat) · text-embedding-3-large (embeddings)
  ├─ Azure Cosmos DB for NoSQL — vector store + chat history
  ├─ Azure Blob Storage — uploaded PDFs
  └─ Application Insights — OpenTelemetry tracing
Azure Container Apps Azure AI Foundry Cosmos DB vector search Quart · Python 3.11 React 19 · Vite · Fluent UI Entra ID managed identity Bicep · azd OpenTelemetry
